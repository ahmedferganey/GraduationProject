67 /*
68  Measure Distance and Print on LCD16x2 using Ultrasonic Sensor and Atmega16 Microcontroller
69 
70  Let’s design an application in which we will find the distance to an object by interfacing
71  ultrasonic module HC-SR04 with AVR ATmega16 and display the distance on 16x2 LCD.
72 
73  Steps of Programming
74 
75  1- ATmega16 microcontroller needs to transmit at least 10 us trigger pulse to the HC-SR04
76  Trig Pin.
77 
78  2- After getting a trigger pulse, HC-SR04 automatically sends eight 40 kHz sound waves
79  and the microcontroller waits for rising edge output at the Echo pin.
80 
81  3- When the rising edge capture occurs at the Echo pin which is connected to an input of
82  ATmega16, start Timer of ATmega16 and again wait for a falling edge on the Echo pin.
83 
84  As soon as the falling edge is captured at the Echo pin, the microcontroller reads the
85  count of the Timer. This time count is used to calculate the distance to an object.
86  Here we are using the input capture mode of ATmega16 on PD6(ICP1) pin as shown in
87  the above interfacing diagram.
88 
89  Calculation (distance in cm) (H1)
90 
91  Sound velocity = 343.00 m/s = 34300 cm/s
92 
93 
94 
95  The distance of Object (in cm) = \mathbf{\frac{Sound Velocity * TIMER}{2}}
96 
97  = \mathbf{\frac{34300 * TIMER}{2}}
98 
99  = \textbf{17150 * TIMER}
100 
101  Now, here we have selected an internal 8 MHz oscillator frequency for ATmega16,
102  with No-presale for timer frequency. Then time to execute 1 instruction is 0.125 us.
103 
104  So, the timer gets incremented after 0.125 us time elapse.
105 
106  = 17150 x (TIMER value) x 0.125 x 10^-6 cm
107 
108  = 0.125 x (TIMER value)/58.30 cm
109 
110  = (TIMER value) / 466.47 cm
111
